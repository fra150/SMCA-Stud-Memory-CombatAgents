## рЯІ† What is BAS?

Current LLMs fail at long context. **BAS changes the paradigm.**

Instead of asking one model to read 500 pages (and inevitably forget), BAS creates **500 specialized agents**, each expert on one page, connected by StudSar as a unified brain. It is a **dynamic cognitive architecture** that combines:

- **Dynamic agent scaling** вАФ N agents = N document segments (not fixed)
- **Semantic selection** вАФ agents chosen by cosine similarity, not position
- **Post-retrieval reasoning** вАФ dedicated executor for numerical aggregation
- **Context-aware filtering** вАФ pre-calculation filtering by transaction type
- **Perfect needle retrieval** вАФ 100% accuracy on facts hidden in 50-segment documents
- **Flawless Multi-Hop Reasoning** вАФ 100% accuracy on correlating independent segments

At its core lives **StudSar Centrale** вАФ the central memory system that transforms static document segments into living associative knowledge, enriched with emotional tags and temporal metadata.

---

## рЯПЫпЄП Architecture вАФ The Five Pillars

### `01` вАФ BAS Engine ¬Ј *The Neural Cortex*

Dynamic agent creation where document complexity dictates agent count. Runs entirely on standard CPU using `sentence-transformers` for embedding generation. Each agent holds exactly one segment, creating true expertise rather than generalized forgetting.

### `02` вАФ Semantic Selection ¬Ј *The Thalamus*

No positional bias. The system calculates cosine similarity between query embedding and segment embeddings, selecting only the most relevant agents for combat. **Impact:** Needle accuracy improved from 50% вЖТ 100%.

### `03` вАФ Post-Retrieval Executor ¬Ј *The Prefrontal Cortex*

A dedicated numerical reasoning layer that separates retrieval from computation. Features multi-language keyword mapping (Italian "entrate" вЖТ English "income"), strict type filtering, and automatic net balance computation (income - expense).

### `04` вАФ StudSar Centrale ¬Ј *The Hippocampus*

The shared memory system connecting all agents as a unified brain. Agents don't just retrieve text вАФ they access living markers with emotional resonance, usage frequency, and temporal decay. Memory coherence monitoring via TMDR (Temporal Memory Decay Regression).

### `05` вАФ Giudice BAS ¬Ј *The Executive Control*

Emergent standards evaluation with confidence scoring. The Judge determines when aggregation requires the Post-Retrieval Executor versus direct answer generation. When uncertainty exceeds thresholds, **Dio** (the User) intervenes.
