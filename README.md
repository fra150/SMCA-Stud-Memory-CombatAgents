## рҹ§  What is SMCA?

Current AI systems forget everything between conversations. **SMCA changes that.**

SMCA is not a tool. Not a chatbot. It is a **collective cognitive architecture** for ML teams that combines:

- **Persistent memory** вҖ” semantic associations that survive across sessions
- **Anticipatory reasoning** вҖ” agents that think ahead before answering
- **Competitive selection** вҖ” only the best response wins the arena
- **Minimal human supervision** вҖ” the system learns when to ask, and when not to

At its core lives **StudSar** (Neural Associative Memory System) вҖ” a memory engine that doesn't retrieve plain text chunks, but builds *living associations* between concepts, enriched with metadata that emulate human memory: emotion, reputation, usage frequency, temporal decay.

---

## рҹҸӣпёҸ Architecture вҖ” The Five Pillars

### `01` вҖ” StudSar В· *The Living Core*

Runs entirely on standard CPU hardware using `sentence-transformers` (all-MiniLM-L6-v2) and advanced text segmentation. Creates intelligent **markers** that retain temporal, emotional, and semantic context вҖ” memory that actually *means* something.

### `02` вҖ” The Arena В· *Combat Between Agents*

No single LLM answers your query. Instead, agents with distinct profiles and specializations вҖ” **Precision, Creativity, Synthesis, Speed** вҖ” compete head-to-head. The reigning Champion is challenged by the next agent; whoever scores higher takes the throne.

### `03` вҖ” The Judge & Emergent Standards В· *Dynamic Evaluation*

The combat rules are not fixed. The Judge вҖ” a meta-cognitive agent вҖ” evaluates responses against **emergent standards** that shift between rounds. Round 1 might weight precision. Round 2, creativity. This ensures no "absolute champion" exists вҖ” only the *best archetype for the current need*.

### `04` вҖ” The Countdown В· *Evolutionary Pressure*

The system operates under **biological urgency**. A dynamic timer progressively reduces available rounds. When time expires, whatever the final Champion holds becomes the definitive answer. Forced convergence under pressure вҖ” just like evolution.

### `05` вҖ” The God Protocol В· *Human Supervision*

*God is the User.* When the Judge cannot confidently declare a winner вҖ” due to responses too similar or too contradictory вҖ” it triggers an **escalation**. The human steps in. Over time, the Judge memorizes human decisions and becomes increasingly autonomous.

## рҹҡҖ Quick Start

### Prerequisites

```bash
# Activate your virtual environment and install dependencies
# Windows PowerShell
.venv\Scripts\Activate.ps1
pip install -r requirements.txt

# Linux / macOS
source .venv/bin/activate
pip install -r requirements.txt
```

### Run the Full Arena Demo

End-to-end orchestration: loads key ML concepts into StudSar, runs arena combats across multiple agents, prints the God Protocol output and final judgment.

```bash
python smca_demo.py
```

> Output state is saved to `/smca_state/` вҖ” visualizations land in `--charts-dir` (default: `graficagent10/`)
